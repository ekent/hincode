## [0.1.0] - 2025-11-04

### Added
- ✅ 基础 Git diff 提取功能
- ✅ OpenAI API 集成
- ✅ CLI 命令行框架 (clap)
- ✅ 彩色终端输出
- ✅ Markdown 输出格式
- ✅ JSON 输出格式
- ✅ 基础代码审查功能
- ✅ 语言检测 (rust, python, javascript, typescript, go, java, c, cpp)

### MVP 限制
- 只审查第一个变更文件
- 暂无配置文件支持
- 暂无交互式模式
- 暂无缓存机制
